# NoSleepBot

Discord-бот с напоминаниями и защитой от сна.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Скопируй `.env.example` в `.env` и вставь свой токен:

```bash
cp .env.example .env
```

Открой `.env` и заполни:
```
DISCORD_TOKEN=твой_токен_здесь
DISCORD_CHANNEL_ID=твой_id_канала
```

## Запуск

```bash
python bot.py
```

## Команды

### Консоль
| Команда | Действие |
|---------|----------|
| `test30` | Тест напоминания 30 мин |
| `test15` | Тест напоминания 15 мин |
| `status` | Статус бота |
| `time` | Московское время |
| `awake on/off` | Защита от сна |
| `!chat test30` | Отправить в Discord |
| `!chat test15` | Отправить в Discord |
| `!chat status` | Статус в Discord |
| `exit` | Остановить бота |

### Discord
| Команда | Действие |
|---------|----------|
| `!status` | Статус бота в чате |

## Расписание

| Время (МСК) | Сообщение |
|-------------|-----------|
| 19:30 | @everyone Через 30 минут кв |
| 19:45 | @everyone Через 15 минут кв |
